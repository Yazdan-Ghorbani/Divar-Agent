# Divar-Agent
Divar AI | دیوار ایجنت  دیوار ایجنت  در دیوار مسائل زیادی وجود دارند، از شما خواسته شده است که جعبه ابزار جادویی خود را بردارید و برای کمک  به حل این مسائل بشتابید! یک جعبه ابزار دیواری باید بتواند مسائل پایین را به راحتی و با دقت بالا حل کند. در  این سوال هشت تسک و هشت فلگ وجود دارد که ایجنت شما باید آن ها را حل کند.
